# .Colab Notebook Information
This section provides an overview and important details regarding the analysis performed in this Google Colab notebook.

# Notebook Purpose:
To perform exploratory data analysis (EDA) on a dataset related to sales and shipments.
To identify key insights regarding business segments, product categories, shipping modes, and profitability.
To demonstrate data manipulation, filtering, sorting, and merging techniques using Pandas.
# Dataset Used:
devp_2024_test-1_dataset - Sheet1.csv: A simulated sales dataset containing transactional data such as order details, product information, and pricing.
Sections Covered:
A. Data Loading & Initial Inspection: Loading data, sampling, and basic information display.
B. Data Description & Visualization: Detailed descriptive statistics and visualizations for both categorical and non-categorical variables.
C. Key Business Insights: Extracting specific insights like most/least preferred ship modes, average delays, largest segments, and product category analysis.
D. Variable Creation & Profitability Analysis: Calculating cost, revenue, and profit, and analyzing profitability by product category and business segment.
E. Data Filtering, Bifurcation, Sorting & Merging: Advanced data manipulation techniques.
# Libraries Used:
pandas for data manipulation and analysis.
plotly.express for interactive data visualizations.
matplotlib.pyplot for static data visualizations.
seaborn for statistical data visualizations.
